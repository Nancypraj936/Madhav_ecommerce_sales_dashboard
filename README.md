# Madhav Ecommerce Sales Dashboard

## Objective
Analyze ecommerce sales data to track performance across states, 
categories, payment modes, and time — built as an interactive 
Power BI dashboard.

## Tools Used
- Microsoft Power BI (DAX, Slicers, Drill-down, Custom Visuals)

## Key Metrics
| Metric | Value |
|---|---|
| Total Sales Amount | 438K |
| Total Profit | 37K |
| Total Quantity Sold | 5,615 |
| Average Order Value | 121K |

## Dashboard Visuals
- KPI Cards — Amount, Profit, Quantity, AOV
- Profit by Month — Bar chart (Jan–Dec)
- Sum of Amount by State — Horizontal bar chart
- Sum of Quantity by Category — Donut chart
- Sum of Quantity by Payment Mode — Donut chart
- Sum of Profit by Sub-Category — Bar chart
- Sum of Amount by Customer Name — Bar chart
- Slicers — Quarter (Q1–Q4), Category filter

## Key Insights
- Maharashtra is the highest selling state
- Clothing dominates category sales (~63%)
- Electronics contributes 21%, Furniture 17%
- COD is most used payment mode (~44%)
- UPI accounts for 21% of payments
- Printers and Bookcases are top profit sub-categories
- Profit dips mid-year (June–September)

## Project Learnings
- Built interactive dashboard with drill-down using parameters
- Created table joins and DAX calculations
- Used 7+ visualization types including donut, scatter, map
- Enabled user-driven filters via slicers
